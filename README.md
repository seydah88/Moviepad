# Moviepad 🎬🥞


A 3-key Breakfast-themed macropad with a rotary encoder that uses KMK firmware!

## Why'd I make it

I wanted to create a keyboard that can be used for controls when you're watching a movie or TV show on your laptop. The waffles and pancakes are inspired by my morning routine, since I often watch a show on my laptop while making breakfast! I also wanted to get experience working with hardware, so I thought this would be a great start!

## BOM
- XIAO RP2040
- 3x Cherry MX Switches
- 3x Blank DSA Keycaps
- Rotary Encoder
- 6x M3 Heatset
- 6x M3x16 Bolt
